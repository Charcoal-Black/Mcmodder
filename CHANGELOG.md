# CHANGELOG

## [2.0.1] - 2026-03-02

**重大改动：现在 Mcmodder 使用 [vite-plugin-monkey](https://github.com/lisonge/vite-plugin-monkey) 并基于 TypeScript 开发！**

### 新增内容

- 现在模组和整合包的主要名称、次要名称和缩写名称也支持单击快捷复制到剪贴板了。

- 在 JSON 导入辅助页面新增“合成表 JSON 管理”。
  
  - 此功能暂无实际用途，且也不兼容百科的主流合成表 JSON 导入格式。

  - 如果一定要说有什么用途的话，不妨瞧瞧[这个](https://github.com/Charcoal-Black/emi)？

  - 先前的 JSON 管理功能更名为“物品 JSON 管理”。

- 完善了“物品 JSON 管理”相关功能。

  - 从模组导入 JSON 时，可以指定导入何种资料类型了（以往只能导入物品/方块）。

  - 完善了备份机制，现在搜索潜在资料时也会定期备份进度，且备份的内容更加完整且易于恢复。

- 在用户个人菜单新增“最近串门”列表。

  - 用户主页右侧新增“收藏此用户”按钮，轻触即可将目标用户在我的“最近串门”列表中置顶！

  - 新增“最近串门跟踪”配置项，启用后所有最近串门的用户（科龄 1 周以上）也会被自动记录在“最近串门”列表中，按照最近一段时间内的串门次数降序排列。

  - 新增“收藏用户样式”配置项，用户可以自由调整被收藏的用户在我的“最近串门”列表中所显示的样式。

- 有新消息提醒时，标签页标题也会附加相应的前缀“[X 条新消息!]”以提醒用户。

- 在模组编辑页面的模组关系中新增快捷排序按钮，轻触即可将本条件组内部的模组排序。优先按照前置-拓展-联动的顺序排列；同类关系的模组按照模组数字编号升序排列。

  - 再轻触一次即可还原为初始顺序。

- 进一步完善了由此脚本重写的“自动链接”功能。

  - 现在对于属于任何已知资料类型（即百科原生的 10 种资料类型、此脚本所预存的 16 种资料类型和所有在用户使用过程中此脚本自动记录的资料类型）的物品，其资料类型图标都会正常显示了！

  - 联网搜索的所有结果，包括模组、整合包、个人作者和开发团队，现在也都可以正常显示了！

### 更改内容

- 将文本介绍内容中的“插件”统一描述为“脚本”。

- 现在无法屏蔽自己了！

- 将“矿词/标签重排版”功能分离成为一个独立的配置项，且默认禁用。

- 调整了审核列表中各审核状态数量统计的显示样式。

- 调整了快捷切换相邻资料 ID 按钮的显示样式。

- 修改了大量夜间模式的样式逻辑。

- “模组添加辅助工具”现在重命名为“展开模组添加分页”。

- 微调了编辑器样式，使之看上去更加自然美观。

- 由于一些性能和兼容性问题，夜间模式的过渡动画被暂时移除了。

- 由于功能不稳定，“Mcmodder风格UI”配置项被暂时移除了。

### Bug 修复

- 公共标语库的链接失效。

- 百科社群的限流机制导致脚本自动检查更新失败。

- 某些情况下编辑可编辑表格中的空单元格后，即使编辑后的内容依然保持为空，也依然会显示为已更改的样式。

- “切换字数统计”按钮显示在预料之外的位置。

- 某些情况下无法横向拖动审核列表。

- 某些情况下编辑器的初始化失败，导致字数实时统计、夜间模式兼容、取消警告延时、快捷定位警告/错误位置等功能不正常工作。

- 在使用自动链接的联网搜索时，搜索结果的主要名称和次要名称相同会导致渲染错误。

- 后台审核界面的功能与排版异常。

- 矿词/标签页的排版异常。

- 链接冲突检查功能高亮错误的位置有误。

- 使用组合中带有 Enter 键的快捷键通过/退回待审项时，二次确认按钮也会被连锁触发。

- 其他一系列拼写错误，以及一系列潜在的逻辑漏洞。

## [1.6] - 2026-01-01

### 新增内容

- 加入 JSON 导入辅助工具，可通过脚本目录或此链接访问。

  - 此工具还非常新，可能会有一些不稳定的地方，遇到问题时请及时反馈！

  - 目前支持的子功能包括：获取模组资料列表、JSON 上传/预览/编辑/下载。

- 重写了“自动链接”功能。现在自动链接功能同时支持本地搜索和联网搜索。

  - 联网搜索与百科原生功能类似。

  - 本地搜索时，脚本将从用户已选中的本地JSON文件中搜索物品，并根据关键词匹配程度降序排列展示。

  - 二者均选中时，两种搜索方式的结果会组合成一个新的结果来显示。

  - 由于数据来源于本地，本地搜索能够几乎一瞬间完成，其速度远快于联网搜索。

  - 目前此功能仅支持搜索和展示物品资料。对于模组/整合包/作者/开发团队，请继续使用百科原生的“自动链接”功能。

- 加入预编辑功能。

  - 现在即使资料被其他编辑者的待审项锁定，也仍然可以编辑资料并提交“预编辑”项。

  - “预编辑”项会被暂存在本地，在此期间用户可以覆盖或删除它们。

  - 脚本会定期检测，直到该资料被解锁为止。脚本自动提交“预编辑”项并告知用户。

- 加入用户快捷切换功能，现在可以在未登录状态和多个曾经登录过的用户身份之间快速切换。

  - 同时移除了先前的“隐身模式”功能。按住 Shift 键时仍可一键切换状态，其效果类似。

- 加入“夜间模式自适应”配置，开启后夜间模式将跟随当前浏览器偏好设置而自动开启或关闭。

- 加入“取消提交警告延时”配置，开启后将取消准备提交编辑时“警告”级别提醒的等待时间。

- 加入“匿名吐槽”配置，开启后创建吐槽时将不再记录创建人。（此项尚未经过测试，不保证能够正常使用）

- 加入了脚本存储缓冲区，以优化脚本设置和计划任务列表的读取速度。

- 加入了分屏排版机制，目前已应用在后台审核页面（分屏显示待审项详情）和合成表编辑页（分屏显示物品检索/选择器）。

### 更改内容

- 调整了后台单项审核页面，现在变得更紧凑了！

  - 同时加入了一些用于审核操作的快捷键，以及相应的配置项。

  - 有效的改动对比信息被尽可能移到了靠前的位置，省去了许多情况下还需要滚动鼠标滚轮的麻烦。

- 现在所有已打开的页面都会实时同步“夜间模式”配置。

- 为脚本设置页的一些配置限定了可输入值的范围，避免意料之外的配置值出现。

- 将“现代化UI”“自动保存修复”“快速提交修复”“模组关系记录器”“经验计算器”“样式管理修复”等特性分别分离成独立的配置项，且现在均默认禁用。

- 现在“图像本地化检测”功能会自动排除百科编辑器不支持上传的图片资源（非 PNG/JPG/JPEG/GIF 格式或文件大小超出 1,024 KB）。

- 将编辑页的“自动链接”“快速提交”快捷键分别分离成独立的配置项，现在可以自由设置。

- “编辑量实时统计”配置更改为接受一个数值，正文字节数超过该值时必须手动触发字数统计，以缓解正文过长时由字数实时统计导致的卡顿现象。

- 现在可以在合成表编辑页中使用上下箭头快捷键来使当前数值 +1/-1（按住 Shift 时为 ×2/÷2）。

- 优化了一批排版（它们的位置现在变得更加灵活了！同时也不太可能会出现某个文字块过大或过小的情况）。

- 移除了所有表态。

- 加入了猎奇。

### Bug 修复

- 即使“编辑量实时统计”配置被禁用，也仍然会统计编辑量。

- 个人目录在某些窄屏环境下仍然可能会超出屏幕右边界。

- 资料正文的代码块排版异常，且不兼容夜间模式。

- 审核列表重排版功能失效。

## [1.5] - 2025-08-02

**重要：此版本包含非常重要的安全性更新，强烈建议所有用户升级至此版本！**

### 新增内容

- 脚本设置加入“资源管理”部分，可以查看脚本在本地保存的数据。
  
  - 现在追踪器会同步记录标语的出现时间和总出现次数，旧的浏览框也被并入新的“资源管理”部分当中。

- 加入了“资料链接检查”功能，开启后会高亮可疑的正文资料链接错误。

- 加入了“表格应用样式”功能，开启后会对正文资料中的表格框线应用主题样式颜色。同时微调了默认的表格框线样式。

- 加入了“今日份好运”功能，开启后脚本会自动获取和记录今日黄历，并显示在 v3 主页。

- 重新加入了“消息中心”按钮，只会在已登录时显示。
  
  - 为避免按钮拥挤过度，回退了先前版本中对 v4 搜索栏位置的改动。

- 加入了一个隐藏功能~

### 更改内容

- 重写了脚本发包逻辑。

  - 旧的 `fetch` 和 `jQuery.ajax` 方法已全部使用油猴 API 的 `GM_xmlhttpRequest` 替代。

  - 现在所有加载脚本的页面共用一个发包间隔计时器，防止多页面同时发送大量请求的情况发生。

  - “自动签到”功能不再需要打开个人主页，“自动查询待审项”功能不再需要打开 MC 百科后台。

  - 重写了脚本的计划任务系统，提升了计划执行精准度与灵敏性，且能够在同一个页面中一次性执行多条相同的计划任务。

- “自动查询待审项”的提醒方式由弹出提示改为弹出弹窗。

- Live2D 百科娘现在可以拖动。

- 微调了模组与整合包页面的宽度 [[#31](https://github.com/Charcoal-Black/Mcmodder/issues/31)]。

- 现在包括留言板编辑器在内的所有编辑器都能够自适应尺寸了。

### Bug 修复

- （百科本体 Bug）光标在列表中时，使用 Ctrl + Enter 快速提交会创建新的列表项。

- 编辑合成表时，产物数量和概率无法设置。

- “无限制留言板”功能无法使用。

- “紧凑化综合子资料”功能导致子资料无法正常显示。

- 夜间模式下 v3 主页搜索栏键入的文字模糊。

- 编辑器自动链接完成后，工具提示有概率保留。

- 自动获取编辑榜数据时，有概率获取到错误的数据。
